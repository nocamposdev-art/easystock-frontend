# EasyStock Frontend

Interfaz web en **React** para la gestión de inventario, conectada a la API EasyStock.

## Funcionalidades
- Login de usuarios
- Visualización de stock
- Filtrado por sucursal y sector según rol
- Diseño responsive con Material UI

## Tecnologías utilizadas
- React
- Material UI
- Axios
- React Router

## Despliegue
Pensado para ser hospedado en **Vercel**, conectado con la API en Render.
